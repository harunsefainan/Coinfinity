# Coinfinity
Coinfinity is a mobile application designed for you to instantly follow the cryptocurrency market, get detailed information about coins and personal account management.

## Project Features

- Firebase Auth
- Firebase Firestore
- Retrofit
- Glide
- Webview
- Google Location
